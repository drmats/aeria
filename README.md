# aeria

[IGC](http://www.ukiws.demon.co.uk/GFAC/documents/tech_spec_gnss.pdf)-based,
simple air time statistics generator.

<br />




## usage

```
$ aeria --help
usage: aeria [-s|--span=y|m|d] [-r|--raw] [--no-total]
```
